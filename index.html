Introduction

This template shows the basic setup of a printable online IEEE formatted paper. This interface provides support for highlighting Communicating Hardware Processes and Production Rules code blocks using a prism-js plugin. It provides support for simple and clean charts using chartist and a few plugins. And it provides support for IEEE formatting.

Usage

Code

inline code may be included using like so *[ L?; R! ]. Block code, must be surrounded with pre tags. The highlight language may be specified by class="language-chp" or class="language-prs". This is inherited, so as long as any parent tag has this class, code will be highlighted with that language.


*[ [ L.f -> R.f+
   [] L.t -> R.t+
   ]; L.e-;
   [~L.f & ~L.t]; R.f-,R.t-;
   L.e+
 ] ||
*[[L.e]; [true->L.f+:true->L.t+]; [~L.e]; L.f-,L.t-] ||
*[[R.f | R.t]; R.e-; [~R.f & ~R.t]; R.e+]
